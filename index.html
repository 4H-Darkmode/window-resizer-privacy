Privacy Policy for Window Resizer

Last Updated: October 28, 2025

Overview

Window Resizer is a simple browser extension that helps you quickly resize your browser window to preset or custom dimensions. Your privacy is important to us.

Data Collection and Storage

This extension stores your custom screen size presets using Chrome's sync storage. This means:

  • Custom presets you create are stored in your Google account's Chrome sync storage
  • If Chrome Sync is enabled, your presets sync across your Chrome browsers where you're signed in
  • The extension developer has no servers and cannot access your data
  • No data is ever transmitted to the extension developer or any third party

What Data is Stored

  • Custom screen size preset names (e.g., "My Laptop")
  • Screen dimensions (width and height values in pixels)
  • Only data you explicitly create by adding custom presets

What Data is NOT Collected

This extension does NOT collect, store, or transmit:

  • Browsing history
  • Personal information
  • Website content
  • Any data outside of your custom presets

Data Usage

  • Data is used solely to provide custom screen size presets functionality
  • Data remains entirely within your Chrome browser and Google account
  • The extension developer has zero access to your data
  • No analytics, tracking, or telemetry of any kind

Permissions Used

  • Storage: Required to save your custom presets

Data Deletion

Your data can be deleted by:

  • Clicking the delete (X) button next to any custom preset
  • Uninstalling the extension (automatically removes all stored data)
  • Disabling Chrome Sync in your browser settings

Third-Party Services

This extension uses Google Chrome's built-in sync storage API to optionally sync your custom presets across devices. This is a standard Chrome feature. No other third-party services, servers, or APIs are used.

Chrome Web Store Compliance

This extension complies with Chrome Web Store policies and the limited use requirements for user data.

Changes to This Policy

We may update this privacy policy from time to time. Updates will be reflected in the extension's listing on the Chrome Web Store.
